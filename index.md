# VEinterpolation

## Installation

You can install the development version of `{VEinterpolation}` from
GitHub:

``` r
devtools::install_github("Epiconcept-Paris/VEBIS_EHR_VE_impact_study_interpolation")
```

## Demo

This package allows to interpolate weekly data from 8-week data from
vaccine effectiveness (VE) data.

See the dedicated vignette for more details:

- [Interpolate Monthly Data to Weekly
  Data](https://epiconcept-paris.github.io/VEBIS_EHR_VE_impact_study_interpolation/articles/interpolate_monthly_data_to_weekly_data.html)
